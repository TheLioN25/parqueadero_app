Nombre del proyecto: Proyecto Parqueadero ERH
Tipo de aplicación: Aplicación web
El Proyecto Parqueadero ERH es una aplicación web diseñada para gestionar de manera eficiente el uso de espacios en un parqueadero. Su objetivo principal es automatizar el registro y control de vehículos, optimizando la administración del servicio.

Problema que resuelve:
La falta de control manual en parqueaderos puede generar errores en el registro de vehículos, pérdida de información y dificultad para conocer la disponibilidad de espacios en tiempo real.
Usuarios del sistema:
•	Administrador del parqueadero
•	Operadores encargados del registro de entradas y salidas
Funcionalidades principales:
•	Registro de usuarios
•	Registro de vehículos
•	Control de entradas y salidas
•	Cálculo del tiempo de permanencia
•	Detección de espacios disponibles

Alcance del proyecto:
El sistema está enfocado en la gestión básica de parqueaderos, permitiendo llevar control de información en tiempo real mediante una interfaz web sencilla y funcional.
